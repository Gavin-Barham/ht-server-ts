# ht-server-ts
A RestAPI backend for a health tracker application specifically for people trying to manage their diabetes. The API is built in Typescript using the Express library and Node.js runtime, along with Docker for containerization and hosting. The API is currently hosted with AWS on an EC2 instance at https://www.health-tracker-api.com.
Documentation can be found at https://health-tracker-api-docs.netlify.app
